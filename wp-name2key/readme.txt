A plugin that provides a Secondlife Name2key Service

== Description ==

This plugin provides a name2key service for secondlife as in secondlife the uuid keys are used for many things from sending commision to another avatar in secondlife, inventory so we 
created this name2key lookup service that does not require huge databases and bots to scan secondlife for keys as we use the secondlife search to get our keys.

== Installation ==

1. Upload `wp-name2key.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[name2key]` in your pages for name2key lookup
